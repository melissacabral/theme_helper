Theme Inspector Plugin for WordPress

Description:

Ever spent too much time trying to figure out which template loads when on your WordPress site? Theme Inspector displays useful information on pages and posts to aid in developing Wordpress themes. Theme Inspector tells you exactly what conditional tags are true, and what files load on each page view.

  • Use In Conjunction with the WP Template Hierarchy Document.
  • Install in the wp-content/plugins directory
  • Theme Inspector is only visible to logged in users.
