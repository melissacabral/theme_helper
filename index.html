Theme Helper Plugin for WordPress

Description:

Ever spent too much time trying to figure out which template loads when on your WordPress site? Theme Helper displays useful information on pages and posts to aid in developing Wordpress themes. Theme Helper tells you exactly what conditional tags are true, and what files load on each page view.

  • Use In Conjunction with the WP Template Hierarchy Document.
  • Install in the wp-content/plugins directory
  • Theme Helper is only visible to logged in users.
